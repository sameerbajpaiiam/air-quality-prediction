# Predictive Environmental Modelling & Multivariate Air Quality Analysis

**High-Performance XGBoost Regression Pipeline**

Architected a robust regression pipeline to forecast critical environmental variables (Temperature and Humidity).

## Key Results
- **R² = 0.91** | **RMSE = 4.10**
- Identified key chemical precursors (CO, NO₂, NMHC) most indicative of environmental shifts and diurnal patterns
- Systematic hyperparameter tuning and feature importance ranking

## Tech Stack
- Python, Pandas, Scikit-learn, XGBoost
- Custom data pipelines (non-standard delimiter handling + mean imputation)
